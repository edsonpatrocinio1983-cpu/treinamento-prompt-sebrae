# 🎓 Engenharia de Prompt: Do Caos ao Resultado

**Treinamento Corporativo Interativo — Sebrae 2025/2026**

Um site de treinamento moderno e interativo sobre Engenharia de Prompt, desenvolvido com HTML5, CSS3 e JavaScript puro.

## 📚 O que está incluído

- ✅ **6 Módulos Estruturados** com conteúdo completo
- ✅ **Checklist Interativo** com pontuação em tempo real
- ✅ **Design Responsivo** (funciona em desktop, tablet e mobile)
- ✅ **Tooltips Explicativos** para termos técnicos
- ✅ **Material de Apoio em PDF** para download
- ✅ **Base Acadêmica Sólida** com referências

## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Baixe o arquivo `index.html`
2. Clique duas vezes para abrir no navegador
3. Pronto! O site funciona offline

### Opção 2: Publicar no GitHub Pages
1. Crie um repositório no GitHub
2. Faça upload do arquivo `index.html`
3. Vá em Settings → Pages
4. Selecione "Deploy from a branch" → "main"
5. Seu site estará online em poucos minutos!
